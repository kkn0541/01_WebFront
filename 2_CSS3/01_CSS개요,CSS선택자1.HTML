css(Cascading style sheets)

        마크업 언어(HTML)가 실제로 화면에 
        표시(모양 , 글씨체 , 색상 , 크기)되는 방법을 기술하는 언어.

css selector(선택자)

        현재 html 문서 내에서 특정한 요소를 선택하는 문자 또는 문자열.
        
        ==스타일을 적용하고자 하는 요소를 선택하는 방법

태그 선택자

    현재 html 문서 내에 같은 태그를 선택하는 선택자.

    [작성법] 
    
    태그명 {
        css 코드;
    }
    ex_
    pre{
        font-size :50px;
    }

아이디 선택자

        html문서 내에서 id속성값이 일치하는 요소를 선택하는 선택자

        [작성법 ]

        #id속성값 {
            css 코드;
        # == id
        }

class 선택자

        html문서 내에서 같은 class 속성값을 가지는 태그를 모두 선택하는 선택자
         클래스 선택자는 중복가능
         [작성법]
         .class 속성값 { css 코드;}

         .==class
